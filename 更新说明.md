# 更新说明

## v1.0.0 (2026-01-12)

### ✨ 核心功能

- **自动渲染**：识别网页中的 PlantUML 代码块并自动渲染为图表
- **多格式输出**：支持 PNG 和 SVG 两种格式
- **智能缓存**：基于代码 hash 缓存，避免重复渲染
- **一键复制**：点击图片即可复制到剪贴板

### 🎨 主题支持

- Default（默认）
- Materia（Material 风格）
- Spacelab（现代简洁）
- Cerulean-Outline（清新线框）
- Vibrant（活力多彩）
- Plain（极简黑白）

### 🌐 平台支持

- **金山文档**：完整测试，主要平台
- **GitHub**：支持标准 Markdown 代码块
- **其他平台**：理论支持所有标准 Markdown 平台（未充分测试）

### 🖥️ 服务器选项

- 官方服务器（推荐）：`https://www.plantuml.com/plantuml`
- Kroki.io服务器（备用）：`https://kroki.io`
- HTTP 备用：`http://www.plantuml.com/plantuml`
- 自定义服务器地址

### 🔍 图片查看器

- **滚轮缩放**：以鼠标位置为中心，10% - 1000% 缩放
- **拖拽平移**：鼠标拖拽移动图片
- **双击切换**：适应窗口/原始大小
- **复制功能**：独立复制按钮，SVG 同时复制为文本和图片

### ⚡ 性能优化

- 初始扫描延迟：100ms
- 防抖延迟：200ms
- MutationObserver 监听动态内容
- 智能缓存机制

### 🎨 视觉设计

- 青草绿配色（#66BB6A → #43A047）
- 专业流程图树状图标
- 圆角现代风格

### ⚠️ 已知限制

- Kroki.io 服务器不支持 materia 主题（自动切换为 default）
- 需要连接 PlantUML 服务器（可自建本地服务器）
